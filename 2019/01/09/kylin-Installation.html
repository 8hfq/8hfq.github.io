Kylin手动安装文档

  1. 下载Kylin 二进制文件拷贝至本地并解压 文件目录/usr/kylin

  2. 配置环境变量 注意修改版本信息

  3. 启动Kylin

遇到的问题以及处理方式

  1. 由于启动kylin需要使用hdfs，所以需要hdfs用户执行 su hdfs
  2. 报错mkdir: cannot create directory ‘/usr/kylin/apache-kylin-2.1.0-bin-cdh57/ext’: Permission denied
    文件目录hdfs用户没有权限 授权kylin目录777
    chmod -R 777 kylin

  3. 报错Failed to find metadata store by url: kylin_metadata@hbase
    检测服务器是否能连接上hbase 执行hbase shell
    若能连接上 可能由于 zookeeper中已存在kylin_metadata 导致报错
    进入zookeeper 删除kylin_metadata 即可

    删除后 重新启动
    运行成功 访问ip:7070/kylin/

Kylin升级文档

  1. 下载安装包
  2. 将新版本的Kylin包解压缩到新文件夹，例如/usr/kylin/apache-kylin-2.2.0-bin（不建议直接覆盖旧实例）
  3. 将旧配置文件（KYLIN_HOME/conf/*）合并到新配置文件中建议不要覆盖新的配置文件。如果修改了tomcat配置（KYLIN_HOME/tomcat/conf/）,请执行相同的操作。
    merge过程中 文件较多 需注意
  4. 停止kylin $KYLIN_HOME/bin/kylin.sh stop

  5. 修改环境变量为新的kylin

  6. 修改web登录帐号

  7. 启动kylin $KYLIN_HOME/bin/kylin.sh start

  8. 升级Hbase协处理器
